# Legion Lab

A tool for MapleStory to help track your Legion's gear and progress.

## Try it

- **Online:** [https://anepicsquirrel-gh.github.io/legion-lab/](https://anepicsquirrel-gh.github.io/legion-lab/) (after you enable GitHub Pages)
- **Locally:** Open `index.html` in a browser, or run `npx serve` and open the URL shown. For **Search** (look up characters from Nexon Rankings) to work without a proxy, run `node server.js` and open http://localhost:3000.

## Features

- Track characters, levels, classes, worlds, and gear (slot + starforce)
- **Search** fills in level/class/world/sprite from Nexon Rankings (uses a CORS proxy when not running the local server)
- Mass import by name list (one name per line)
- Gear presets, set bonuses, and legion block icons
